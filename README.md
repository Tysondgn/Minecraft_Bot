# Minecraft_Bot
A Minecraft_bot created using NPM mineflayer
how to install.

1. Clone Repository
2. npm update
3. npm index.js

Before running file you need to add the port inside index.js from your Minecraft.

if index.js run propperly congratulations🎉 your bot jined minecraft world
you can command it in the chat "follow" to follow you and "Stop" to stop it from following you it has memory and his own taks to survive so yo dont need to always command it.
